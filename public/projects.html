Projects

Below are some side projects that I have worked on this summer. Most of these projects do not relate but one underlying theme between them would be some form of computer intelligence. Some projects are inspired by online courses I have taken on Coursera or Jovian.ml . My lucid dreaming site was initated when I got my first introduction to simplified sentiment analysis while the lane detection project followed a Coursera course that focused on CNNs.

Chess

I enjoy playing chess in my freetime so I decided to code up my own game. There are variant board positions available to try out. [Magnus Carlsen vs. Li Chao 2015]

Number Recognition

Utilized the public mnist datasets to train my model. Used pygame for the user interface. The digits have to be very carefully drawn for it to really work.

A* Search

Visualization of the A* Search Alg. The directional nature is so interesting to manipulate in contrast to Dijkstra's or others. Viewing differences between euclidean and manhattan heuristics are subtle but significant

Lane detection

Utilizing the open-cv library, here is a really basic lane detection program. Next is detecting horizons so that the road detection can recognize lane shapes more fluidly.

Dream Log [Link]

Currently in progress. I'm fascinated by lucid dreaming so I built a site centered around it. It also doubles as a side project for NLP as each week I will run a simple script that analyzes user entries in the community forum for basic emotions, general topics, and frequency of items. This can be found under the trends section.

Space Cruiser

Basic game created with pygame. Hitting planet reduces lives. Fun introduction to graphics and game logic. Sound effects too!
